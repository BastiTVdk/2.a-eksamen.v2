Toppunktsformlen: Find x-koordinaten ($h$)

Vi ved fra dine noter, at $f(x) = ax^2 + bx + c$ (1. rep) og $f(x) = a(x-h)^2 + k$ (2. rep).

Trin 1: Startpunktet

Vi sætter de to udtryk lig hinanden: $ax^2 + bx + c = a(x-h)^2 + k$

Hvad skal vi gøre ved højre side for at komme videre?

Trin 2: Udvidelsen

Korrekt! Vi folder parentesen ud: $ax^2 + bx + c = a(x^2 + h^2 - 2xh) + k$

Hvad er det næste logiske træk?

Trin 3: Sammenligningen

Flot! Vi får: $ax^2 + bx + c = ax^2 + ah^2 - 2ahx + k$

Kig nu kun på de led, der indeholder $x$ (dvs. $bx$ og $-2ahx$). Fordi udtrykkene er ens, må de led være ens.

Hvordan isolerer vi $h$ fra ligningen $b = -2ah$?

BEVIS GENNEMFØRT! 🎉

Du har vist, at $h = -\frac{b}{2a}$

(Dette er x-koordinaten for toppunktet $T_p$)
